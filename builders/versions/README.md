# Pixel Parent Builder Guidelines — past versions

The **current** guidelines are the live ones in [`builders.md`](../../builders.md) at
the repo root (rendered at [pixelparents.org/builders](https://pixelparents.org/builders)).

This folder is the archive of **past, superseded versions**, one file per version:

- [`v0.1.md`](./v0.1.md) — the original proposed guidelines (pre‑ratification).

When a new version is ratified, snapshot the outgoing `builders.md` into this folder
as `vX.Y.md` before editing the live file, so every prior version stays linkable from
the "view past versions" link in the page subtitle.
